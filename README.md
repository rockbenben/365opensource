# 365 开源计划

> 一个人 + AI，一年 300+ 个开源项目。

## 这是什么

365 开源计划是一个 AI 原生开发实验——用 AI 大幅压缩开发时间，在一年内产出 300+ 个实用的开源工具。

所有项目完全开源，所有人都可以免费使用。

## 项目范围

**✅ 接受的类型：**
浏览器扩展、Web 在线工具、Python 脚本 / 自动化、油猴脚本、CLI 命令行工具、静态站点 / H5 页面

**❌ 不接受的类型：**
需要后端服务器的 SaaS、需要数据库的完整应用、移动端 App、需要持续运维的项目

## 如何参与

### 提交需求（免费）

👉 [点击提交需求](https://my.feishu.cn/share/base/form/shrcnI6y7rrmlSjbzkYXh6sjmzb)

任何人都可以提交需求，不需要登录。所有项目完全开源，所有人都可以使用。

### 赞助优先

扫码赞助 100 元起（100 × N 倍 = 更高优先级），付款后加微信发截图确认。

赞助者权益：

- 🔥 **优先开发**：金额越高，优先级越高
- 💬 **专属社群**：加入赞助者微信群，开发窗口期内直接沟通
- 📛 **项目署名**：你的名字出现在对应项目 README 的赞助者栏
- 🟢 **GitHub 贡献**：加入 Contributors 列表，点亮你的 GitHub 贡献热力图

> 即使你不是开发者，也可以通过赞助开源项目在 GitHub 上留下自己的足迹。

### 开发规则

- 每个项目投入时间控制在 **24 小时**以内
- 赞助项目在窗口期内可直接沟通调整需求
- 超出窗口期后进入社区维护状态，欢迎 PR
- 不承诺长期维护，但会保持代码可用
