> One person + AI, 300+ open source projects in one year.

## What is This

The 365 Open Source Plan is an AI-native development experiment — using AI to dramatically compress development time and ship 300+ practical open source tools in a single year.

All projects are fully open source and free for everyone to use.

## Project Scope

**Accepted types:**
Browser extensions, Web tools, Python scripts / automation, Userscripts, CLI tools, Static sites / H5 pages

**Not accepted:**
SaaS requiring backend servers, Full applications requiring databases, Mobile apps, Projects requiring ongoing maintenance

## How to Participate

### Submit a Request (Free)

👉 [Submit via GitHub Issues](https://github.com/rockbenben/365opensource/issues/new?labels=request&title=[Request]+Your+project+idea)

Anyone can submit a project idea — just open a GitHub Issue. Please include:

- **What** the tool should do (one sentence)
- **Why** you need it (the problem it solves)
- **Type** (browser extension, web tool, CLI, etc.)

All projects are fully open source and free for everyone.

### Sponsor for Priority

Sponsor to get your project prioritized. Contact [@rockbenben](https://github.com/rockbenben) via GitHub for sponsorship details.

Sponsor benefits:

- **Priority development** — higher amount = higher priority
- **Project credit** — your name in the project README's sponsor section
- **GitHub contribution** — added to the Contributors list, lighting up your GitHub contribution graph

> Even if you're not a developer, you can leave your mark on GitHub by sponsoring open source projects.

### Development Rules

- Each project is limited to **24 hours** of development time
- Sponsored projects allow direct communication during the development window
- After the window, projects enter community maintenance — PRs welcome
- No long-term maintenance commitment, but code will be kept functional
